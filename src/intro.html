Welcome to CTArcade!

The CTArcade project was initiated by Dr. Ben Bederson. Initial research was performed by the Human Computer Interaction Lab (HCIL) at the University of Maryland. The HCIL worked to develop a session's worth of activity that provided challenges for gauging the critical thinking skill of a group of children through the KidsTeam project initative. The initial findings suggested that a critical thinking tool would be of benefit to teachers and students. A complete description of the project is provided in [Ahn. J, 2012], which summarizes the challenges and motivation of this project.

The primary motivation behind this project is to develop a framework for a crticial thinking tool that teaches users through a simple interface that utilizes existing skills internalized in the user's mind, as opposed to a complex graphical programming tool. Rather than having users program logical rules directly in a bottom up approach, users will "discover" and rank rules to train an "artificial intelligence" (AI) that will play for them on a tournament server composed of other similarly trained entities. Playing on the server will result in a score and challenging individual AI will provide users with a history of game play to review. These two artifacts are intended to provide motivation for the user to reflect and improve on their AI by learning from their experiences.

Below is a list of frequently asked questions about using CTArcade and about the project in general. If you have any problems using CTArcade or if you need any additional information, please send an email to: ctarcade@gmail.com

What is the CTArcade?

Design a system that will help to develop a non-programmer's computational thinking (CT) skill through game play while providing motivation for social and competitive interactions.
